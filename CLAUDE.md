# Claude Instructions

이 저장소의 기준 지침은 `AGENTS.md`입니다.

Claude로 작업할 때도 먼저 `AGENTS.md`를 읽고 그 내용을 따르십시오. 특히 다음 원칙은 현재 구현과 직접 연결됩니다.

- GM 화면은 TV/대형 디스플레이 사용을 우선 고려합니다.
- GM 화면은 상단 바가 아니라 하단 floating bar를 사용합니다.
- 모바일 세로 모드는 정식 지원 대상입니다.
- 슬라이드의 `data-title`은 공통 화면 제목이며, 본문 첫 제목이 중복되면 런타임에서 숨깁니다.
- 순서/흐름 다이어그램은 위에서 아래로 흐르는 세로 구조와 CSS/SVG 벡터 화살표를 사용합니다.
- 텍스트 화살표(`→`, `=>`, `->`)는 화면용 마크업에 직접 넣지 않습니다.
- 검증은 `node --check app.js`와 `npm test`를 기본으로 합니다.

지침을 수정할 때는 이 파일에 내용을 중복 복사하지 말고 `AGENTS.md`를 업데이트하십시오. 이 파일은 Claude용 브리지 문서로 유지합니다.
